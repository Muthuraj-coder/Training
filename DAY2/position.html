CSS Position Property Examples

Static Position (default):
This box is static (default).
Relative Position:
This box is moved 30px right and 10px down from its normal position.
Absolute Position:
This box is absolutely positioned 20px from top and left of its nearest positioned ancestor.
This is the containing block (relative).
Fixed Position:
This box is fixed to the bottom-right of the viewport.
Scroll the page to see the fixed box stay in place.
Sticky Position:
This box will stick to the top of its container when you scroll.
Scroll inside this box to see sticky in action.
